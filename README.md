# Guild Wars 2 Emote Wheel

An Emote Wheel Overlay for Guild Wars 2.  
Works by using makro-style key simulation to send the emote commands to the chat box.  


![](https://yeleha.co/2JzjSoL) 

![](https://i.imgur.com/niOCxYc.gif)


## Features / Usage

* All 24 Emotes in a nice and clear layout - Accessible using **Alt+C**
* Support for emote targeting - Toggle mode using **Shift**
* Support for emote syncing - Toggle mode using **Control**



## Installation

* Download the [latest release](https://github.com/InventivetalentDev/GuildWars2EmoteWheel/releases/latest)
* Unzip the file
* Run the **GW2 Emote Wheel.exe** in the extracted folder
